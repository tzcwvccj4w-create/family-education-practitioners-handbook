# GitHub Pages V2.0 上传说明

本压缩包是纯静态网站，不需要安装任何程序。

## 上传位置

把压缩包内的全部文件和文件夹上传到仓库根目录（main 分支）：

- index.html
- 404.html
- .nojekyll
- assets/
- read/
- skills/
- downloads/
- citation/

不要删除现有的 Book、Skills、Resources、Figures、README、LICENSE、CITATION.cff 和 CHANGELOG。

## 上传方法

1. 打开 GitHub 仓库首页；
2. 点击 Add file → Upload files；
3. 将本包内的文件拖入上传区域；
4. Commit message 填：`Build GitHub Pages website V2.0`；
5. 点击 Commit changes；
6. 等待约 1—3 分钟，刷新官网。

官网地址：
https://tzcwvccj4w-create.github.io/family-education-practitioners-handbook/

## 说明

- `index.html` 存在后，GitHub Pages 会优先展示 V2.0 网站，不再把 README 当首页。
- 章节页是从正式书稿自动提取的文字版；原书插图和复杂版式仍以 PDF 为准。
- Skill 页面支持一键复制完整指令。
